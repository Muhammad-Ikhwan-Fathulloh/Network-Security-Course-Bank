Apa yang Membuat Halimun-Proxy Berbeda?

  • End-to-End Encryption: Payload dienkripsi dengan AES-256-CBC dari browser hingga backend
  • Replay Attack Prevention: Setiap request memiliki Nonce unik yang divalidasi di memory
  • Camouflage URLs: URL routing diubah menjadi random segments
  • SSRF Protection: Mencegah akses ke internal network (127.0.0.1, 192.168.x.x)
